# Personal Portfolio Website

## Overview
A modern, responsive portfolio website showcasing my skills, projects, and professional journey as a Full Stack Developer. Built with HTML5, CSS3, and JavaScript.

Live Link: https://shantanudwivedi1.github.io/My_Portfolio/

## Features

- Responsive design that works on all device sizes
- Dynamic background slideshow with smooth transitions
- Interactive UI elements with hover effects
- Skills and project showcase sections
- Testimonials carousel
- Contact form with validation
- Smooth scrolling and animations

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Libraries**: Font Awesome, Google Fonts, EmailJS
- **Tools**: Git, VS Code, Chrome DevTools

## Project Structure
```
My_Portfolio/
│
├── index.html          # Main HTML file
├── style.css           # Main stylesheet
├── script.js           # JavaScript functionality
├── images/             # Image assets
│   └── img.png         # Profile image
│
└── README.md           # Documentation
```

## Getting Started

### Prerequisites
- Modern web browser
- Text editor (VS Code recommended)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/My_Portfolio.git
   ```
2. Navigate to the project directory and open `index.html` in your browser

## Customization

### Colors
The color scheme can be modified in `style.css`:
```css
:root {
    --primary-color: #007ACC;
    --accent-color: #64FFDA;
    /* Add/modify colors as needed */
}
```

### Content
Update personal information in `index.html`:
- Profile details
- Skills
- Projects
- Contact information
